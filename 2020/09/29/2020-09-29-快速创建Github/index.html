快速搭建Github

创建Github账号

先点击github

其中Sign up是注册，Sign in是登入。

其中注册的时候，建议Username不要填得很怪，后期你会后悔的 = W =

然后注册后，需要在自己注册所用邮箱中去确认激活下，才拥有Star和fork一系列功能。

库与文档的操作

创建库

如图所示，到你个人的页面里去，可能新手创建后页面有些区别，但大同小异。去点击那个“NEW”，或者new repositories（其他界面）

转入创建库的界面

给库取一个贴切的名字（这个以后可以改）然后，点Create repository

在库中创建文档

点击Create a new file

写文档&保存

其中写内容。

点commit

提交 pull request

pull request 是可以去看别人写的代码文档，然后自己提出一些优化和改进的，可以在这里与作者互动，交流心得，如果作者愿意，会将你的code融合到他自己的code里面，以后如果有面试，你说起自己在某个开源大项目里面有作者的pull ，也是种加分项，比如曾经github上开源过美国阿波罗火箭上天的代码·····

github上面还有”fork“，”Git“等功能，也是十分强大，有兴趣的话可以去查查各大论坛帖子，相信你会在github上面发掘出更多东西。

感谢观看
